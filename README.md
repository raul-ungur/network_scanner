# Network Scanner

LAN scanner developed in Python with Flask and Nmap.

## Requirements

- Python 3.11+
- Nmap installed and available in your PATH

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/network-scanner.git

Go to the folder:

cd network-scanner

Create the virtual environment:

python -m venv .venv

Activate the virtual environment:

Windows:

.venv\Scripts\activate

Linux/Mac:

source .venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Start the server:

python app.py

Open your browser:

http://127.0.0.1:5000

Click "Scan Network" to detect devices on your local LAN.
